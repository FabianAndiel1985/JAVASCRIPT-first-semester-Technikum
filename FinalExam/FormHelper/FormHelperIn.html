Aufgabe

  • Beide Textarea-Felder des untenstehenden Forumlars sollen, wenn der User einen Text eingibt analysiert werden. Sobald die Schluesselwoerter "iaeo", "unido" oder "osze" eingegeben werden, sollen diese mit ihren Grossbuchstaben ersetzt werden.
    Beispiel: Eingabe: "Ich arbeite in der iaeo" wird ersetzt mit: "Ich arbeite in der IAEO" (40 Punkte)
  • Verwenden Sie ein custom HTML Attribut "tokenize" um anzuzeigen, dass das Feld diese Funktionalitaet erhalten soll (15 Punkte).
  • Verwenden Sie Attribut-Werte fuer das custom HTML Attribut "tokenize" um die Schluesselwoerter fuer jedes Feld anzugeben (15 Punkte).
